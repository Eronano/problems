# 题目描述

给定 $n$, $k$，计算 $\left(\sum_{i=1}^ni^k\right)\bmod p$，即 $1^k+2^k+...+n^k$ 除 $p$ 的余数，其中 $p=49999$ 为质数。

# 输入格式

第一行为一个整数 $t~(1\leq t\leq 500)$。

接下来 $t$ 行每行有两个整数 $n~(1\leq n< 2^{64})$, $k~(0\leq k<2^{64})$，含义同描述。

# 输出格式

输出 $t$ 行, 每行输出一个整数, 第 $i$ 行为第 $i$ 次计算的结果。

# 输入输出样例

```input1
3
1 3
2 0
100 1
```

```output1
1
2
5050
```
