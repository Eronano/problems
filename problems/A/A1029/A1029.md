# 题目描述

给定一个 $1 \times n$ 的矩形格条，你需要对每个格子涂上红色或者蓝色。但是两个红色的格子不允许相邻，请问共有多少种涂色的方案？

因为答案可能很大，你只需要输出答案对 $998244353$ 取模的结果即可。

# 输入格式

一行，一个整数 $n~(1 \leq n \leq {10}^{18})$。

# 输出格式

一行，一个整数，代表答案。

# 输入输出样例

```input1
3
```

```output1
5
```

```input2
2333333
```

```output2
394054407
```

# 说明/提示

【样例解释 #1】

$5$ 种合法的涂色方法分别为：红蓝红，红蓝蓝，蓝红蓝，蓝蓝红，蓝蓝蓝。
