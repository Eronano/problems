# 题目描述

输出一组数据的中位数。

# 输入格式

第一行，一个整数 $n~(1 \leq n \leq 1 \times {10}^5)$。

第二行，$n$ 个整数 $a_i~(1 \leq a_i \leq 1 \times {10}^5)$。

# 输出格式

一行，输出中位数。

# 输入输出样例

```input1
4
1 2 3 4
```

```output1
2.5
```

# 说明/提示

对于 $50 \%$ 的评测用例，$1 \leq n, m \leq 1 \times {10}^3$。

对于所有评测用例，$1 \leq n, m \leq 1 \times {10}^5$。
