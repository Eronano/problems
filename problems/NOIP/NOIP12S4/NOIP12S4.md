# 题目描述

求关于 $x$ 的同余方程 $ax \equiv 1 \pmod {b}$ 的最小正整数解。

# 输入格式

一行，包含两个正整数 $a,b$，用一个空格隔开。

# 输出格式

一个正整数 $x_0$，即最小正整数解。输入数据保证一定有解。

# 输入输出样例

```input1
3 10
```

```output1
7
```

# 说明/提示

【数据规模】

对于 $40 \%$ 的数据，$2 \leq b \leq 1000$；

对于 $60 \%$ 的数据，$2 \leq b \leq 5 \times {10}^7$；

对于 $100 \%$ 的数据，$2 \leq a,b \leq 2 \times {10}^9$。
