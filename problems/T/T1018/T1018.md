# 题目描述

有 $n$ 个石子，两个人轮流从中取走石子，每次最多取 $m$ 个，最少取 $1$ 个，取完者获胜。

请问先手方有没有必胜的策略。

# 输入格式

第一行，一个正整数 $t(1 \leq t \leq {10}^5)$，代表共有 $t$ 组数据。

接下来 $t$ 行，每行两个整数 $n_i,m_i(1 \leq m_i \leq n_i \leq {10}^9)$，代表共有几个石子及每次最多取几个石子。

# 输出格式

对于每组数据，在一行内输出其答案（若先手方有必胜的策略则输出 `YES`，否则输出 `NO`）。

# 输入输出样例

```input1
2
13 2
20 3
```

```output1
YES
NO
```
