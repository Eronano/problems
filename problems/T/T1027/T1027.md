# 题目描述

给定一棵树，对任意结点 $u$ 定义 $f(u)$ 为其它所有结点到 $u$ 结点的距离之和，请你求出 $\min\limits_{u \in V}(f(u))$。

# 输入格式

第一行，一个整数 $n~(1 \leq n \leq {10}^5)$，代表树上结点个数。

接下来 $n-1$ 行，每行两个整数 $u,v~(1 \leq u,v \leq n)$，代表结点 $u,v$ 之间有一条边。

# 输出格式

一个整数，代表结果。

# 输入输出样例

```input1
7
1 3
2 3
6 4
7 4
4 5
3 5
```

```output1
10
```
