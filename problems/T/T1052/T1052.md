# 题目描述

给定若干个 $n \times m$ 的网格，请你计算出每个网格中有多少个矩形。

例如下图为一个 $2 \times 2$ 的网格，其中有 $9$ 个矩形。

![](file://image.png)

# 输入格式

第一行，一个整数 $q~(1 \leq q \leq 2 \times {10}^5)$，代表共有多少个网格。

接下来 $q$ 行，每行两个整数 $n,m~(1 \leq n,m \leq {10}^9)$，代表网格长宽。

输出的结果对 $998244353$ 取模。

# 输出格式

$q$ 行，每行一个整数，代表每个网格中有多少个矩形。

# 输入输出样例

```input1
2
2 3
233 666
```

```output1
18
65501853
```
